# CoolBlue App

This app demonstrates to search products by using a mocked CoolBlue API and lists them in a grid recyclerview.

- Hilt
- Coroutines
- Flow
- MVVM
- Retrofit
- Glide
- Clean Architecture
- Sdp & Ssp
- Custom Pagination
